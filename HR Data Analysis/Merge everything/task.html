Description

Your datasets are ready for merging! It will make data analysis much more straightforward.

Use concatenation to generate a dataset with information from both offices. Use the left merging by index to merge the previously created dataset with the HR's dataset. When joining, generate a column containing information about each row's origin. Keep only those employees' records that are present in both datasets.

Recall the data structure of tables:

  1. A_office_data.xml and B_office_data.xml:
    • number_project — number of projects an employee has worked on;
    • average_monthly_hours — typical workload per month in hours;
    • time_spend_company — how many years an employee has worked in the company;
    • Work_accident — whether an employee has had an injury at work;
    • promotion_last_5years — whether an employee has had any promotions during the last five years;
    • Department — employee's department;
    • salary — employee's salary rate;
    • employee_office_id — employee's ID.
  2. hr_data.xml:
    • satisfaction_level — how well an employee performs their job;
    • last_evaluation — the last evaluation score of an employee;
    • left — whether an employee has left the company;
    • employee_id — employee's ID in the company.
Objectives

It's time to create one comprehensive dataset! The first three steps are the same as for Stage 1:

  1. Load all three datasets. Remember, data is stored in XML format;
  2. Explore the data — check the axes, get the shapes of the DataFrames, use df.info() to check whether there are null values and review the data types;
  3. Reindex all three datasets in the same way as in the previous stage;
  4. Use the concat() function from pandas to generate a unified dataset for offices A and B;
  5. Use df.merge() to carry out the left merging of the unified office dataset with HR's dataset. Merge both datasets by index — use left_index and right_index parameters. Set indicator=True in df.merge(). For the final table, leave only those employees whose data is contained in both datasets;
  6. Drop the employee_office_id, employee_id columns and the new column that contains a row source;
  7. Sort the final dataset by index;
  8. Print two Python lists: the final DataFrame index and the column names. Output each list on a separate line.
